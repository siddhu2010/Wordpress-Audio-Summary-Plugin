# 🎙️ WordPress Audio Summary Plugin

Transform your WordPress blog posts into engaging audio experiences with a lightweight and customizable Text-to-Speech (TTS) plugin.

This plugin automatically adds an elegant audio player to your blog posts, allowing visitors to listen instead of read—improving accessibility, engagement, and overall user experience.

---

## ✨ Features

- 🎧 Convert blog posts into audio
- 🗣️ Supports multiple Text-to-Speech providers
- ⚡ Lightweight & performance-friendly
- 📱 Fully responsive design
- 🎚️ Playback speed control
- 🔊 Volume control
- ⏯️ Play, Pause & Stop controls
- ⌨️ Keyboard shortcuts
- 🎨 Customizable UI
- ⚙️ Easy WordPress integration
- ♿ Accessibility focused
- 🔌 API-agnostic architecture

# 🚀 Why I Built This

Modern users don't always have time to read lengthy articles.

This plugin enables website visitors to consume content through audio, making blogs more accessible while increasing user engagement and time on page.

Instead of relying on heavy server-side processing, the plugin uses Text-to-Speech APIs to generate spoken content with minimal performance impact.

# 🔄 How It Works

```
Visitor Opens Blog Post
            │
            ▼
Plugin Extracts Article Content
            │
            ▼
Content is Cleaned & Prepared
            │
            ▼
Sent to a Text-to-Speech Provider
            │
            ▼
Interactive Audio Player Appears
            │
            ▼
Visitor Listens to the Article
```

---

# 🌐 Compatible Text-to-Speech APIs

The plugin has been designed to work with different Text-to-Speech services depending on your requirements.

Examples include:

- ResponsiveVoice
- ElevenLabs
- Google Cloud Text-to-Speech
- Amazon Polly
- Microsoft Azure Speech
- Any REST-based TTS API

You can easily replace the provider based on pricing, voice quality, language support, or business requirements.

# ⚙️ Installation

1. Download the plugin.
2. Upload it to your WordPress Plugins directory.
3. Activate the plugin.
4. Configure the settings.
5. Open any blog post and start listening.
